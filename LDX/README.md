# 跨学科知识图谱可视化系统 🌐

一个用于探索和可视化跨学科概念关联的交互式 Web 应用。

## ✨ 功能特性

- 🎯 **智能搜索**：快速查找相关概念
- 🎨 **多学科筛选**：按学科筛选展示概念
- 🔗 **交互式图谱**：点击、拖拽、缩放、悬停交互
- 📊 **信息展示**：详细的概念和关联信息
- 💾 **数据导出**：支持 PNG 图片和 JSON 数据导出
- 📱 **响应式设计**：适配各种屏幕尺寸
- 🌓 **主题切换**：支持浅色/深色主题

## 🚀 快速开始

### 方式一：直接打开（最简单）

1. 双击 `index.html` 文件
2. 在浏览器中打开应用

### 方式二：使用本地服务器

```bash
# 安装依赖（可选）
npm install

# 启动本地服务器
npm start

# 或使用Python启动
python -m http.server 8000
```
